# LiveView for Claude — 사용량 라이브뷰

**Claude 사용량을 실시간으로 보여주는, 항상 화면 위에 떠 있는 Windows 미니 위젯.**

하루에도 몇 번씩 claude.ai 설정을 열어 사용량을 확인하고 있다면 — 이제 그럴 필요가 없어요. LiveView는 화면 구석에 조용히 떠서 5시간 세션·주간·모델별(Opus, Fable 등) 사용량을 미니멀한 링 게이지로 보여줍니다. 1분마다 자동 갱신되고, 리셋까지 남은 시간도 함께 표시돼요.

> 🎨 **개발자가 아닌 아티스트가 만들었습니다.** 이 앱은 서울의 디자인 스튜디오 [레임(LEIM)](https://www.leim.kr)을 운영하는 아티스트 [로렌킴](https://www.instagram.com/artist_lorenkim)이 Claude와의 대화만으로 기획·디자인·개발했습니다. OAuth 인증부터 WPF 링 게이지까지 모든 코드는 Claude(Cowork)가 작성했고, 디자인 시스템은 로렌킴의 것입니다.

## 주요 기능

- **항상 위 미니 창** — 원하는 곳에 끌어다 두면 위치를 기억해요
- **링 게이지** — 5시간 세션, 주간 전체, 모델별 주간 한도(Fable, Opus 등 자동 표시)
- **리셋 카운트다운** — 각 한도가 언제 풀리는지 한눈에
- **주간/야간 테마** — 세이지 그린 라이트 모드, 웜 브라운 다크 모드 (◐ 버튼)
- **공식 OAuth 로그인** — Claude Code와 같은 인증 방식. 쿠키·비밀번호 불필요, 토큰 자동 갱신
- **언어 자동 감지** — Windows 언어에 따라 한국어/영어
- **무설치** — 작은 스크립트 파일 두 개가 전부. 관리자 권한도 필요 없어요
- **전부 열람 가능한 코드** — PowerShell 스크립트라 누구나 열어서 확인할 수 있어요

## 설치 (2분)

1. 최신 릴리스를 내려받아 원하는 폴더에 압축 해제 (예: `문서\LiveView`)
2. **`ClaudeUsageWidget.bat`** 더블클릭
3. **[1. 브라우저에서 로그인 열기]** → Claude 로그인 → **[허용]** → 표시되는 코드 복사 → 위젯에 붙여넣고 **[2. 연결]**. 끝!

선택: **`autostart-on.bat`**을 한 번 실행하면 로그인할 때마다 자동으로 떠요 (`autostart-off.bat`으로 해제).

## 개인정보 · 보안

- 인증 토큰은 **내 컴퓨터에만** 저장됩니다 (`%APPDATA%\ClaudeUsageWidget\config.json`)
- 위젯은 **Anthropic 서버(api.anthropic.com, claude.ai) 외에는 어디와도 통신하지 않아요**
- 수집·분석·외부 전송 일절 없음. 스크립트를 직접 열어 확인하세요

## 요구 사항

- Windows 10 또는 11 (PowerShell 5.1 기본 내장)
- Claude 계정 (Pro/Max 등 사용량 한도가 있는 플랜)

> 🍎 **macOS 버전**(메뉴바 형태)도 준비 중이에요 — 스타(⭐)를 눌러두시면 소식을 받아보실 수 있어요.

## 문제 해결

- **"인증이 만료됐어요"** → ⚙에서 다시 연결
- **"요청이 많아 잠시 쉬어가요"** → 자동으로 재시도하니 그대로 두면 돼요
- 그 외 → 스크립트 폴더의 `debug_log.txt` 내용과 함께 이슈를 남겨주세요

## 고지

이 앱은 Anthropic과 무관한 **비공식** 커뮤니티 도구입니다. Anthropic의 OAuth 엔드포인트를 사용하며, 해당 API는 예고 없이 변경될 수 있습니다.
